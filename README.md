# [게시판 프로젝트]

- **프로젝트 명**: Spring Boot와 MySQL을 이용한 게시판 구현
- **프로젝트 기간**
  - 2021.07.16 - 2021.07.26

- **게시판 기능** 
  - CRUD 기능 (게시글 등록, 조회, 수정, 삭제)
  - paging 처리
  - 조회수 기능

- **개발 환경**
  - Java 8 
  - Spring Boot 2.3.5
  - mysql 
  - mybatis
  - apache tomcat 

- **사용 기술**
  - bootstrap
  - thymeleaf
